# Task 4 - Smart Product Listing Website with Local Storage

## Project Title

Smart Product Listing Website with Local Storage

## Description

This project was developed as part of the ApexPlanet Web Development Internship Task-4. The application combines advanced HTML, CSS, and JavaScript concepts to create a responsive and interactive web experience.

The project includes a product listing page with filtering and sorting features, a portfolio section, and a local storage-based task/notes management system.

## Features

### Personal Portfolio Section

* About Me
* Skills
* Projects
* Contact Information
* Responsive Navigation Bar

### Product Listing Page

* Product Cards
* Product Images
* Product Categories
* Product Prices
* Product Ratings
* Product Descriptions

### Filtering Features

* Filter Products by Category
* Filter Products by Price Range

### Sorting Features

* Sort by Price (Low to High)
* Sort by Price (High to Low)
* Sort by Rating

### Local Storage Integration

* Add Tasks or Notes
* Delete Tasks or Notes
* Save Data in Browser Local Storage
* Retrieve Saved Data After Refresh

### Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Compatible
* Built using Flexbox and CSS Grid

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API
